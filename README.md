# cs7290_project

Some resources:
https://github.com/ChampSim/ChampSim

## Instructions
Run `get_dpc3_traces.sh` to get the trace data.
```sh
cd scripts
./get_dpc3_traces.sh
```

Run `trace_to_dataset.sh` to transform trace to train/validation data. No need to regenerate if it is already there
```sh
cd scripts
./trace_to_dataset.sh
```
